# @helium/treasury-management-sdk

TypeScript SDK for the [treasury-management program](../../programs/treasury-management), which runs a bonding curve that swaps DNT (MOBILE/IOT) for HNT from a sub-DAO treasury at a formula-derived price. This is how hotspot earners exit their DNT into HNT.

Note: You should not use anything other than the current constant product swap. There may be slippage issues with other formulas.
