# @helium/tuktuk-dca-sdk

TypeScript SDK for the [tuktuk-dca program](../../programs/tuktuk-dca), which schedules recurring DCA swaps that the [tuktuk cranker](https://github.com/helium/tuktuk) executes via [`tuktuk-dca-service`](../tuktuk-dca-service).

This is no longer used, as we are no longer burning Helium Mobile Revenue
